## Key Notes

HTML is a markup language used to structure content on the web.

An HTML document consists of:

* `<!DOCTYPE html>` → defines HTML5
* `<html>` → root element
* `<head>` → metadata (not displayed)
* `<body>` → visible content

The `<title>` appears in the browser tab.

The `<h1>` tag represents the main heading of the page.

Backend servers often generate HTML responses that are sent to the browser.

Request → Server → HTML Response → Browser Rendering
