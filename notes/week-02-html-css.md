## Key Notes

HTML is a markup language used to structure content on the web.

An HTML document consists of:

* `<!DOCTYPE html>` → defines HTML5
* `<html>` → root element
* `<head>` → metadata (not displayed)
* `<body>` → visible content

The `<title>` appears in the browser tab.

The `<h1>` tag represents the main heading of the page.

Backend servers often generate HTML responses that are sent to the browser.

Request → Server → HTML Response → Browser Rendering

## Key Notes

HTML tags define the structure and meaning of content.

Common tags:

* `<h1>` → main heading
* `<p>` → paragraph text
* `<a>` → link to another page
* `<img>` → display an image

Links connect documents:

* `<a href="page.html">Link</a>`
* Clicking a link sends a request to another resource

Images are external resources:

* `<img src="image.jpg">`
* The browser sends a request to the server to load the image

Backend servers handle these requests and return the requested resources.

Request → Server → Resource (HTML/Image) → Browser Rendering
